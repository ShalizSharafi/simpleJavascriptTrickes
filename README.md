## 📋 About

This project is a front-end practice exercise centered around implementing JavaScript popups and modal dialogs. It uses a travel booking website as the visual context — complete with hotel search, destination cards, and navigation — but the core learning goal is the popup/modal behavior triggered by user interaction.

## ✨ Features

- **Popup modal** — triggered on page load or by user action, with a dismiss option ("No, thanks")
- **Destination cards** — showcasing sample travel destinations with pricing
- **Navigation bar** — with icons for Hotels, Flights, and Rentals
- **Responsive layout** — uses a viewport meta tag for mobile compatibility

## 🛠️ Built With

- HTML5
- CSS3
- Vanilla JavaScript

## 🧠 Concepts Practiced

- Showing and hiding DOM elements with JavaScript
- Handling button click events

## 🚀 Getting Started

To run locally, simply clone the repo and open `index.html` in your browser — no build tools or dependencies needed.

```bash
git clone https://github.com/shalizsharafi/simpleJavascriptTrickes.git
cd simpleJavascriptTrickes
open index.html
```
